## 🚀 Как добавить текстуры

### Шаг 1: Создайте PNG файлы
- Размер: 16x16, 32x32 или 64x64 пикселей
- Формат: PNG с альфа-каналом (RGBA)
- Стиль: Пиксель-арт или Minecraft-подобные

### Шаг 2: Поместите в правильную папку
```bash
# Блок ID 3 (STONE)
cp stone.png textures/inventory/blocks/3.png

# Предмет ID 103 (Iron Pickaxe)
cp pickaxe.png textures/inventory/items/103.png
```

### Шаг 3: Запустите игру
```bash
./gradlew run
```

Система автоматически:
- Загрузит текстуры
- Кеширует их
- Отобразит в инвентаре

## 💻 Код для использования

```java
import hrd.h4rdykrft.render.InventoryTextureManager;

// Загрузить текстуру блока
int blockTexId = InventoryTextureManager.loadBlockTexture(3);

// Загрузить текстуру предмета
int itemTexId = InventoryTextureManager.loadItemTexture(103);

// Получить кешированную текстуру
int cachedId = InventoryTextureManager.getItemTextureId(103);

// Получить статистику
System.out.println(InventoryTextureManager.getStats());

// Очистить кеш при выходе
InventoryTextureManager.clearCache();
```
