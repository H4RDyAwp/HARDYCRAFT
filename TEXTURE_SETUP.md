### 2. **Создан InventoryTextureManager**

Новый класс для управления загрузкой и кешированием текстур:

```java
// Загрузка текстур блоков
int blockTexId = InventoryTextureManager.loadBlockTexture(3); // STONE

// Загрузка текстур предметов
int itemTexId = InventoryTextureManager.loadItemTexture(103); // Iron Pickaxe

// Получение кешированной текстуры
int cachedId = InventoryTextureManager.getItemTextureId(103);

// Статистика
System.out.println(InventoryTextureManager.getStats());

// Очистка при выходе
InventoryTextureManager.clearCache();
```

## 📋 Как добавить текстуры

### Шаг 1: Подготовить изображение
- Формат: PNG (с прозрачностью)
- Размер: 16x16, 32x32 или 64x64 пикселей
- Центрировано в файле
- Прозрачный фон

### Шаг 2: Поместить в правильную папку

**Для блока:**
```bash
cp your_texture.png textures/inventory/blocks/ID.png
```

**Для предмета:**
```bash
cp your_texture.png textures/inventory/items/ID.png
```

### Шаг 3: Система загрузит автоматически

При запуске игры система:
- ✅ Найдёт текстуру по ID
- ✅ Загрузит её в OpenGL
- ✅ Кеширует для производительности

## 📝 Требования к текстурам

### Технические характеристики:
- **Формат:** PNG
- **Цвета:** RGBA (8 бит на канал)
- **Прозрачность:** Альфа-канал (обязателен)
- **Размер:** 16x16, 32x32 или 64x64
- **Сжатие:** PNG без потерь

### Требования к содержимому:
- Центрировано в изображении
- Прозрачный фон
- Небольшие отступы по краям
- Пиксель-арт стиль или текстуры в стиле Minecraft
